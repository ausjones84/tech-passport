## [2.0.0] - 2026-04-06

### Added
- AI Professor Coach with real-time personalized feedback
- 6 production-grade cloud engineering scenarios:
  - Kubernetes Cluster Failure
  - Terraform Deployment Error
  - AWS IAM Permission Issue
  - Production API Outage
  - CI/CD Pipeline Failure
  - Infrastructure Scaling Problem
- Job Readiness Score (rubric-based, 0-100)
- Visual Architecture Diagrams for each scenario
- AI Mock Interview module (Professional & Elite)
- Income Strategy Lab (Elite tier)
- AI-powered Resume Analyzer
- AI-powered LinkedIn Profile Optimizer
- Voice support across all modules
- Global career access pathways
- Stripe-powered subscription billing (Starter $50/mo, Professional $100/mo, Elite $150/mo)
- 5-day free trial with no credit card required

### Infrastructure
- Deployed on Vercel (auto-deploy on main branch)
- Supabase for auth + database
- OpenAI GPT-4 for all AI features
- GitHub Actions CI/CD pipeline
