## Core Features

**AI Professor Coach** - Get instant personalized feedback on every answer like having a senior cloud engineer reviewing your work 24/7.

**Real Engineering Scenarios** - Practice Kubernetes failures, Terraform errors, AWS IAM issues, API outages, CI/CD failures, and scaling problems.

**Visual Architecture Diagrams** - Understand how production systems are truly designed with detailed visual breakdowns.

**Job Readiness Score** - A rubric-based 0-100 readiness tracker measuring real progress toward specific engineering roles.

**AI Mock Interviews** - Full AI-powered mock interview sessions simulating real technical interview conditions.

**Income Strategy Lab** - Strategies to maximize your earning potential as a cloud engineer.

**Global Career Access** - Built so engineers anywhere can prepare for global tech roles and remote cloud jobs.

- Auth: Supabase Auth
- Payments: Stripe
